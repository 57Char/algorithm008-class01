## 二分查找代码模板
[模板](https://shimo.im/docs/hjQqRQkGgwd9g36J/read)

```go
func binarySearch(array []int, target int) int {
	left, right := 0, len(array)-1
	for left <= right {
		mid := left + (right-left)>>1
		if array[mid] == target {
			// find the target
			return mid
		} else if array[mid] < target {
			left = mid + 1
		} else {
			right = mid - 1
		}
	}
	return -1
}
```
